Getting Started

Learn how to integrate and use Bootstrap gradient backgrounds in your project.

Installation

Include the gradient CSS file after Bootstrap:

<link href="bootstrap.min.css" rel="stylesheet">
<link href="gradient-backgrounds.css" rel="stylesheet">

Basic Usage

Apply gradient classes to any element:

<div class="bg-linear-primary">
  Linear Primary Gradient
</div>

<div class="bg-radial-success">
  Radial Success Gradient
</div>

Responsive Gradients

Gradients that adapt to different screen sizes.

Mobile Specific: Use .bg-linear-mobile-vertical and .bg-radial-mobile-center for mobile-optimized gradients. 
@media (max-width: 768px) {
  .bg-linear-mobile-vertical {
    background: linear-gradient(to bottom, var(--bs-primary), var(--bs-primary-bg-subtle));
  }
}

Customization

Learn how to customize and extend the gradient system.

Custom CSS Variables

Override Bootstrap's CSS variables to customize gradient colors:

:root {
  --bs-primary: #your-color;
  --bs-primary-bg-subtle: #your-subtle-color;
}

/* Gradients will automatically use your custom colors */

Creating Custom Gradients

.my-custom-gradient {
  background: linear-gradient(135deg, #667eea, #764ba2) !important;
}

.my-radial-gradient {
  background: radial-gradient(circle, #ff6b6b, #feca57) !important;
}
